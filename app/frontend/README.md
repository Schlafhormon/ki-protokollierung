# GremienPilot Frontend

React-/TypeScript-Frontend für GremienPilot. Hinweise zur Installation, zum
lokalen Start und zur Konfiguration stehen in der
[Projekt-README](../../README.md).

## Entwicklung

```bash
npm install
npm run dev
```

Weitere Prüfungen:

```bash
npm run lint
npm run typecheck
npm test
npm run build
```
